# Mine Sweeper Game
* Windows: Get softwares like Dev c++. Compile. And run.
* Ubuntu
        
        1. Open Terminal.
        2. Run g++ -o Mine_Sweeper mine_sweeper.cpp
        3. Then run ./Mine_Sweeper
        
Bingo! Your game runs now. In game Instructions will show as you play.
